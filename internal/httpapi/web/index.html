What you share with peers

These toggles control how much of your local index is exposed to other SwartzNet peers via the sn_search BEP-10 extension. None of these settings affect the local HTTP API (which always stays on localhost) or the Layer-D DHT publisher (which is controlled separately by the --no-dht flag at startup).
